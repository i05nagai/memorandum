## Amazon Simple Notification Service
Amazon SNS enables message filtering and fanout to a large number of subscribers, including serverless functions, queues, and distributed systems. 
Pub/Sub service.

## Concepts
* Event sources
* Message Filtering
    * topic based filtering
        * Subscriber can choose the topic to subscribe
    * attribute based filtering
        * SNS requires the publisher to set message attributes and each subscriber to set a subscription attribute (a subscription filter policy).
* Message fanout
* Message Encryption
* Message Privacy
* Mobile notifications
