## dep

Install

```
go get -u github.com/golang/dep/cmd/dep
```


## Usage

```
$ mkdir -p $GOPATH/src/github.com/me/example
$ cd $GOPATH/src/github.com/me/example
$ dep init
$ ls
Gopkg.toml Gopkg.lock vendor/
```
