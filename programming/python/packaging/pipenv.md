## Pipenv
Officially recommended, python package mangaer.

## CLI

```
pipenv install
```

* `--dev`
    * install dev package
    * by default, pipenv installs only non-dev package
* `--rm`

## Usage
Create a new project

```
pipenv --python 3.7
```

Run command in virtualenv envrionment

```
pipenv run python foobar.py
```
