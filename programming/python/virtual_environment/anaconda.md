## Anaconda
* condaはpackage manager
* anacondaはplatform

## CLI
create environment

```
conda create -n env_name packages
```

activate environment

```
souce activate env_name
source deactivate env_name
```

install package

```
conda install package
```


```
conda env list
```

Delete environment

```
conda remove -n py2 --all
```
