## Binary Search

* $n \in \mathbb{N}$,
* $a_{0} \le \cdots \le a_{n-1}$,
    * $a_{i} \in \mathbb{N}$,
* $k \in \mathbb{N}$,

$lb = -1$,
$ub = n$,

$mid = \lfloor (lb + ub) / 2 \rfloor$,

if $a_{mid} \le lower_bound$, then

$ub - lb = 1$
Suppose that binary search work correct up to $ub - lb = n-1$.
