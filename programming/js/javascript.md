## javascript

## Syntax

* await
    * expression
        * A Promise or any value to wait for.
    * rv
        * Returns the fulfilled value of the promise, or the value itself if it's not a Promise.

```
[rv] = await expression;
```

* async


```
async function [name]([param1[, param2[, ..., paramN]]]) {
   statements
}
```

* yield
    * [rv] = yield [expression];

```
[rv] = yield [expression];
```
