## Grub2
As the computer starts, GRUB 2 either presents a menu and awaits user input or automatically transfers control to an operating system kernel.

## Configuration

* `/boot/grub`
    * GRUB2 modules
    * `/grub.cfg`
        * menu file
* `/etc/grub/grub.cfg`
* `/etc/grub.d/`
    * main GRUB2 scripts
* `/etc/default/grub`
    * user customization files
