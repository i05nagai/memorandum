## ldd
print shared object dependencies.

## CLI

```
ldd [OPTION]... FILE...
```

* `-d, --data-relocs`
    * process data relocations
* `-r, --function-relocs`
    * process data and function relocations
* `-u, --unused`
    * print unused direct dependencies
* `-v, --verbose`
    * print all information
