## groupadd
The groupadd command creates a new group account using the values specified on the command line plus the default values from the system.


## CLI

```
groupadd <options> GROUP
```

* `-f, --force`
* `-g, --gid GID`
* `-K, --key KEY=VALUE`
* `-p, --password PASSWORD`
* `-r, --system`
