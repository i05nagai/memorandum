## rsyslogd

## CLI

```
rsyslogd [ -d ] [ -D ] [ -f config file ] [ -i pid file ] [ -n ] [ -N level ] [ -C ] [ -v ]
```

## Usage

## configuration
* `/etc/rsyslog.conf`
    * base configuration
    * global configuration
* `/etc/rsyslog.d/*.conf`
    * configuraiton for specific application
