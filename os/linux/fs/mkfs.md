## mkfs
build a Linux filesystem.
Use `mkfs.<type>`.

## Usage

```
mkfs [options] [-t type] [fs-options] device [size]
```

```
# alias for "mke2fs -t fs-type"
mkfs.ext2
mkfs.ext3
mkfs.ext4
```
