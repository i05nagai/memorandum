## vim keymap


## default keymapping
Show keymap `:map`.

Show current leader `:let mapleader`.
If this is undefined, defaut leader `\ ` is used

### move
* `gg`
    * ファイルを移動したり移動した場合にカーソルの位置を戻す
* `<C-o>`
    * 元いた場所に戻る
* `<C-i>`
    * 元いた場所に進む
* `<C-^>`
    * 直前に開いていたバッファに戻る
* `<leader>`
    * by default `\ `
