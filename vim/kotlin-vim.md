## kotlin-vim

## Install

以下をvimrcに追加して、`:NeoBundleInstall`

```
NeoBundleLazy 'udalov/kotlin-vim', {
    \ 'autoload' : {'filetypes' : 'kotlin'}
    \ }
```

## Usage
機能は以下だけなので、特に設定項目はない。

* syntax hilight
* syntactic support
* indentation
