## vim-scala

## Install
For NeoBundle,

```vim
  NeoBundleLazy 'derekwyatt/vim-scala', {
        \ 'autoload' : {'filetypes' : 'scala'}
        \ }
```

`:NeoBundleInstall`

## Commands
Sort imports

```
:SortScalaImports
```
