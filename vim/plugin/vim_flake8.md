## Configuration

```vim
" default
let g:flake8_quickfix_height=5
" show (default)
let g:flake8_show_quickfix=1
" show
let g:flake8_show_in_gutter=1
" don't show (default)
let g:flake8_show_in_gutter=0
" don't show (default)
let g:flake8_show_in_file=0
" (default)
let g:flake8_max_markers=500
```
