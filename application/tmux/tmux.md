## tmux

## Status line

* `#(date)`
    * shell command
* `#I`
    * window index
* `#S`
    * session name
* `#W`
    * window name
* `#F`
    * window flags
* `#H`
    * Hostname
* `#h`
    * Hostname, short
* `#D`
    * pane id
* `#P`
    * pane index
* `#T`
    * pane title
