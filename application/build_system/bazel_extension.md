## Bazel Extension
Extend `BUILD` file with extensions for

* Macro
    * macro returns rule
    * If you want to reuse rules or declare common rules through different `BUILD` files, you can define specific Macro
* Rules
* Depsets
* Aspects
* Repository Rules
