## ctags

## Install

For ubuntu 16.04,

```
apt-get install exuberant-ctags
```

## CLI

```
ctags
```

* `--language`
* `-f`
* `--extra`
    * `f`
        * include filename

## Usage
Create tags from current directory.


```
ctags -R --extra=+f
```
